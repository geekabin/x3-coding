**翻译源代码**

```
Subprog LECTEXTRA(VALEUR,TABLE,CHAMP,CLE1,CLE2)
Variable Char    VALEUR()
Value    Char    TABLE, CHAMP, CLE1, CLE2
Local    Char    ACTION(20)
Local Integer    OK
If GPRODUIT=1 : Gosub MODIF_CLE : Endif : # Géode
OK = 1
ACTION = "LECTEXTRA" : Gosub ACTIONB
If !OK : End : Endif
If clalev([F:AXX])=0 : Local File ATEXTRA [AXX] : Endif
Read [AXX]AXX0 = TABLE;CHAMP;GLANGUE;CLE1;CLE2
If fstat=0
 VALEUR = [F:AXX]TEXTE
Else
 Read [AXX]AXX0 = TABLE;CHAMP;GLANDEF;CLE1;CLE2
 If fstat=0
  VALEUR = [F:AXX]TEXTE
 Else
  VALEUR = ""
 Endif
Endif
ACTION = "ClOSETEXTRA" : Gosub ACTIONB
End
```



**调用实例**

```
Variable Char UNIT
Variable Char VALEUR()
UNIT = [M:PSH1]PUU(NOLIGN-1)
#Call XUNIT_FORMAT_CHI(UNIT) FROM XUF
Call LECTEXTRA(VALEUR,"TABUNIT","DES",UNIT,"") From ATEXTRA
```



**参数解释**

```
VALEUR,TABLE,CHAMP,CLE1,CLE2

返回值，表，字段，值1，值2

TABLE,CHAMP,CLE1,CLE2 对应 ATEXTRA 表  CODFIC,ZONE,IDENT1,IDENT2

```



**相关SQL**

```
SELECT TEXTE_0,IDENT1_0 FROM ATEXTRA WHERE CODFIC_0='TABUNIT' AND ZONE_0='DES' AND LANGUE_0='CHI' AND IDENT1_0 = 'SET'
```
